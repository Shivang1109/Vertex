# PrivateIDE Keyboard Shortcuts

## Global Shortcuts

| Shortcut | Action | Context |
|----------|--------|---------|
| `Ctrl/Cmd + 1` | Switch to Dev Mode | Any tab |
| `Ctrl/Cmd + 2` | Switch to Research Mode | Any tab |
| `Ctrl/Cmd + 3` | Switch to Chat | Any tab |
| `Ctrl/Cmd + 4` | Switch to Vision | Any tab |
| `Ctrl/Cmd + 5` | Switch to Voice | Any tab |
| `Ctrl/Cmd + 6` | Switch to Tools | Any tab |

## Dev Mode Shortcuts

| Shortcut | Action |
|----------|--------|
| `Ctrl/Cmd + E` | Explain code |
| `Ctrl/Cmd + D` | Generate docstring |
| `Ctrl/Cmd + B` | Debug code |
| `Ctrl/Cmd + R` | Refactor suggestions |
| `Ctrl/Cmd + Enter` | Run selected action |
| `Escape` | Cancel generation |

## Research Mode Shortcuts

| Shortcut | Action |

| `Ctrl/Cmd + Enter` | Ask question |

| `Escape` | Cancel processing |

## Chat Mode Shortcuts

## Monaco Editor Shortcuts (Dev Mode)

The code editor supports all standard VS Code shortcuts:

| Shortcut | Action |
|----------|--------|
| `Ctrl/Cmd + F` | Find |
| `Ctrl/Cmd + H` | Find and replace |
| `Ctrl/Cmd + /` | Toggle comment |
| `Alt + Up/Down` | Move line up/down |
| `Shift + Alt + Up/Down` | Copy line up/down |
| `Ctrl/Cmd + ]` | Indent line |
| `Ctrl/Cmd + [` | Outdent line |
| `Ctrl/Cmd + Shift + K` | Delete line |
| `Ctrl/Cmd + Shift + L` | Select all occurrences |

## Tips

1. **Copy results quickly**: After generation, results are automatically focused - just press `Ctrl/Cmd + C`
2. **Quick navigation**: Use number shortcuts (1-6) to switch between modes rapidly
3. **Escape to cancel**: Any running generation can be stopped with `Escape`
4. **Enter to submit**: In most input fields, `Enter` submits the form
